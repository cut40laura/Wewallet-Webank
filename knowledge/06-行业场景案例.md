# 第六优先级：行业场景案例

> **用途**：客户说"我是做五金批发的"，Agent 能问得更像真客户经理。
> **说明**：以下案例基于微业贷公开服务数据（客户主要集中在批发零售、制造业、建筑业等）及行业通行风控逻辑整理。
> **更新日期**：2026-05-20

---

## 一、批发零售

### 行业特征
- 微业贷客户中占比最高的行业之一
- 资金周转快，季节性明显
- 上下游账期错配常见

### 常见融资用途
- 旺季备货（节假日、电商大促前）
- 填补上下游账期缺口
- 拓展新品类/新供应商
- 支付仓储租金

### 常见材料
| 材料 | 说明 |
|------|------|
| 进货单/采购合同 | 与上游供应商的订货记录 |
| 销售流水 | POS流水、对公流水、微信支付宝收款 |
| 库存清单 | 当前备货量与资金缺口对应 |
| 下游订单 | 大客户预订单、框架协议 |

### 常见风险点
- 库存积压滞销
- 大客户拖款（商超、国企账期长）
- 价格波动（大宗商品批发）
- 电商冲击导致线下萎缩

### 合理追问问题
> "您主要批发什么品类？旺季是什么时候？现在备了多少货？"
> "最大的客户是谁？占您销售额多少？账期多久？"
> "这次贷款主要是进什么货？有确定的下游买家吗？"
> "如果这批货卖不出去，您还有其他资金来源还款吗？"

### 还款来源判断
- 主要看：销售回款速度、库存周转率、大客户稳定性
- 健康指标：库存周转天数<60天，应收账款账期<90天

## 二、餐饮

### 行业特征
- 现金流以日常营收为主
- 受选址、口味、口碑影响大
- 人工和租金是两大固定成本
- 疫情后复苏行业，部分银行有专项支持

### 常见融资用途
- 开新店/装修升级
- 采购食材/酒水
- 支付厨师/员工工资
- 应对淡季周转

### 常见材料
| 材料 | 说明 |
|------|------|
| 门店租赁合同 | 证明经营稳定性 |
| 近半年流水 | 美团/饿了么流水+对公/个人卡流水 |
| 食材采购单 | 与供应商的订货记录 |
| 员工工资表 | 核实人工成本 |

### 常见风险点
- 食品安全事故
- 选址失误导致客流不足
- 厨师/核心员工流失
- 平台抽成高（外卖占比大时）
- 证照不全（消防、卫生、排污）

### 合理追问问题
> "您这家店开了多久？是单店还是连锁？"
> "堂食和外卖占比多少？主要靠哪个平台？"
> "每个月租金和人工大概多少？占营业额多少比例？"
> "这次贷款是用于开新店还是老店周转？新店选址定了吗？"

### 还款来源判断
- 主要看：日均流水、翻台率、客单价稳定性
- 健康指标：租金+人工占营收<50%，月流水能覆盖月供2倍以上

---

## 三、物流运输

### 行业特征
- 重资产行业（车辆、油费、保险）
- 高度依赖下游合同（工厂、电商、冷链需求）
- 油价波动直接影响成本
- 部分涉及网络货运平台

### 常见融资用途
- 购置/维修运输车辆
- 支付油费、过路费
- 司机工资
- 车辆保险/年检

### 常见材料
| 材料 | 说明 |
|------|------|
| 运输合同/协议 | 与货主/平台的承运协议 |
| 车辆登记证/行驶证 | 证明运力 |
| GPS轨迹/运单记录 | 证明实际运营 |
| 油费/ETC流水 | 证明运营支出 |

### 常见风险点
- 单一大客户依赖
- 交通事故/车辆损坏
- 油价大幅上涨
- 平台压价（货拉拉、满帮等）
- 司机难招/人力成本上升

### 合理追问问题
> "您有多少台车？是自有还是挂靠？"
> "主要给哪些客户运货？有没有长期合同？"
> "运费的结算周期是多久？有没有被拖欠过？"
> "这次贷款是买车还是日常周转？如果是买车，有对应的运输合同吗？"

### 还款来源判断
- 主要看：运输合同金额与稳定性、车辆利用率、运费回款速度
- 健康指标：车辆月均行驶里程稳定，运费回款周期<30天

## 四、制造加工

### 行业特征
- 微业贷服务的重要客群（占比高）
- 设备投入大，生产周期长
- 原材料价格波动敏感
- 下游多为B端大客户

### 常见融资用途
- 采购原材料（钢材、塑料、电子元器件等）
- 设备维护/小范围升级
- 支付工人工资
- 垫资生产（大客户账期长）

### 常见材料
| 材料 | 说明 |
|------|------|
| 采购合同/订单 | 原材料采购记录 |
| 生产订单 | 下游客户的生产指令 |
| 设备发票 | 证明产能和投入 |
| 工人工资表 | 核实用工规模 |

### 常见风险点
- 原材料价格暴涨
- 下游大客户砍单/转移订单
- 环保政策影响（限产、搬迁）
- 应收账款周期长（制造业常见3-6个月）
- 技术迭代导致设备贬值

### 合理追问问题
> "您主要生产什么产品？给哪些客户供货？"
> "原材料最近价格稳定吗？采购周期多长？"
> "下游客户一般多久付款？最长的账期多久？"
> "这次贷款是用于哪一笔订单的生产？合同签了吗？"

### 还款来源判断
- 主要看：在手订单金额、应收账款质量、产能利用率
- 健康指标：产能利用率>70%，订单交付周期与回款周期匹配

## 五、建材五金

### 行业特征
- 与房地产/基建周期高度相关
- 赊销普遍，账期较长
- 库存占压资金大
- 客户多为装修公司、工地包工头

### 常见融资用途
- 备货（瓷砖、管材、五金件等）
- 垫资供货（工地常要求先货后款）
- 支付厂家货款
- 旺季前囤库存

### 常见材料
| 材料 | 说明 |
|------|------|
| 厂家代理/经销合同 | 证明正规渠道 |
| 工地供货单 | 送货签收记录 |
| 装修公司合作协议 | 下游合作证明 |
| 库存盘点表 | 证明备货需求 |

### 常见风险点
- 房地产下行导致需求萎缩
- 工地拖欠货款（甚至项目烂尾）
- 库存跌价（新品替代、装修公司倒闭）
- 客户集中度（依赖一两个大工地）

### 合理追问问题
> "您主要做批发还是零售？客户是装修公司还是个人？"
> "最近房地产市场不太好，对您的生意影响大吗？"
> "工地拖欠货款的情况多吗？最长拖过多久？"
> "这次贷款进的货，有确定的买家吗？还是备库存？"

### 还款来源判断
- 主要看：下游装修公司/工地的付款信誉、库存周转、房地产区域景气度
- 健康指标：应收账款账期<90天，库存品类为畅销品

## 六、服装贸易

### 行业特征
- 季节性强（春夏秋冬订货会）
- 款式迭代快，库存风险高
- 退货率较高
- 部分涉及外贸（汇率风险）

### 常见融资用途
- 季节性备货（换季前大量进货）
- 参加订货会/展会
- 支付加工厂货款
- 应对退货导致的资金缺口

### 常见材料
| 材料 | 说明 |
|------|------|
| 订货合同 | 与品牌方/工厂的订货协议 |
| 展会参展合同 | 证明市场拓展投入 |
| 销售流水 | 线上线下渠道流水 |
| 库存清单 | 当前库存结构与库龄 |

### 常见风险点
- 过季库存积压
- 款式判断失误导致滞销
- 退货率超预期
- 外贸客户汇率损失/拒收
- 电商冲击导致线下关店潮

### 合理追问问题
> "您做服装多久了？主要做批发还是零售？自有品牌还是代理？"
> "现在是什么时候？是换季备货还是日常周转？"
> "过季的库存一般怎么处理？占比多少？"
> "这次贷款进的货，款式定了吗？是补畅销款还是试新款？"

### 还款来源判断
- 主要看：当季销售率、库存库龄、退货率历史数据
- 健康指标：当季新品售罄率>60%，库存库龄<90天占比>70%

## 七、电商

### 行业特征
- 平台依赖度高（淘宝、京东、拼多多、抖音等）
- 推广费用占比高
- 账期特殊（平台结算周期、预售模式）
- 退货率高于线下

### 常见融资用途
- 大促备货（618、双11、年货节）
- 预付货款（工厂要求定金）
- 平台推广费/广告费
- 日常周转（平台账期错配）

### 常见材料
| 材料 | 说明 |
|------|------|
| 平台店铺后台截图 | 销售额、订单量、退货率 |
| 工厂采购合同 | 备货来源 |
| 推广费账单 | 直通车/钻展/千川等 |
| 平台结算账单 | 证明回款周期 |

### 常见风险点
- 平台规则变化（流量成本上升）
- 大促销量不及预期导致库存积压
- 差评/违规导致店铺降权
- 供应商断货/质量问题
- 仅退款、高退货率侵蚀利润

### 合理追问问题
> "您主要做哪个平台？做了多久？店铺等级怎么样？"
> "这次贷款是为哪个大促备货？预计备货金额多少？"
> "平台结算周期多久？有没有压款？"
> "您这个类目的退货率大概多少？利润能覆盖吗？"

### 还款来源判断
- 主要看：店铺月均销售额、平台结算稳定性、大促ROI历史
- 健康指标：推广费占销售额<20%，退货率<行业均值

## 八、供应链上下游小微企业

### 行业特征
- 依附核心大企业（国企、央企、上市公司）
- 账期被动（常被拖欠60-180天）
- 议价能力弱
- 但订单稳定性相对较高

### 常见融资用途
- 垫资生产/供货（核心企业账期长）
- 原材料采购
- 工资发放（等核心企业回款）

### 常见材料
| 材料 | 说明 |
|------|------|
| 与核心企业的供应合同 | 证明订单来源 |
| 中标通知书 | 国企/央企采购项目 |
| 送货/验收单 | 证明已履约 |
| 应收账款确认函 | 核心企业确认应付金额 |

### 常见风险点
- 核心企业拖欠/延期付款
- 核心企业更换供应商
- 账期越拖越长
- 对单一大客户依赖度过高

### 合理追问问题
> "您主要给哪家/哪些企业供货？合作多久了？"
> "他们的付款周期是多久？有没有拖欠过？"
> "如果这家客户不续约了，您还有其他客户吗？"
> "这次贷款是为了给哪家客户垫资供货？合同签了吗？"

### 还款来源判断
- 主要看：核心企业信用等级、历史付款准时率、应收账款账期
- 健康指标：核心企业为国企/央企/上市公司，历史付款准时，单一客户占比<50%

## 九、通用追问框架（所有行业适用）

无论客户说什么行业，都可以按以下框架追问：

```
1. 经营历史
   - 做这行多久了？
   - 之前是打工还是一直自己做？

2. 经营现状
   - 现在有几个店/几个客户/几台设备？
   - 月销售额大概多少？
   - 旺季和淡季差多少？

3. 资金需求
   - 这次缺多少钱？
   - 用在什么地方？（具体到供应商/用途）
   - 什么时候需要用？

4. 还款来源
   - 打算怎么还这笔贷款？
   - 主要是靠哪部分收入还？
   - 如果这部分收入没来，有备用方案吗？

5. 风险排查
   - 有没有其他贷款？每个月还多少？
   - 下游客户有没有拖欠货款的？
   - 最近生意有没有受大环境什么影响？
```

## 十、行业速查表

| 行业 | 核心风险点 | 关键材料 | 还款来源 |
|------|-----------|----------|----------|
| 批发零售 | 库存积压、大客户拖款 | 进货单、销售流水 | 销售回款 |
| 餐饮 | 选址失误、人工租金高 | 租赁合同、平台流水 | 日营收 |
| 物流运输 | 油价、事故、平台压价 | 运输合同、运单 | 运费结算 |
| 制造加工 | 原材料涨价、账期长 | 采购/生产订单 | 订单回款 |
| 建材五金 | 房地产下行、工地拖欠 | 供货单、经销合同 | 工地结算 |
| 服装贸易 | 过季库存、退货率高 | 订货合同、库存清单 | 当季销售 |
| 电商 | 平台依赖、推广成本高 | 后台截图、推广账单 | 平台结算 |
| 供应链企业 | 核心企业拖欠、依赖度高 | 供应合同、验收单 | 核心企业回款 |
