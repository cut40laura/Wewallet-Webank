# 第二优先级：材料清单

> **用途**：客户问"没有合同怎么办、能不能先预审、流水看哪个账户"时用。
> **更新日期**：2026-05-20
> **核心原则**：微业贷为纯信用线上产品，**申请阶段无需提交纸质材料**；以下材料主要用于**客户经理尽职调查、用途核实、风控追问**场景。

---

## 一、基础身份材料

| 材料 | 用途 | 说明 |
|------|------|------|
| **法人身份证** | 实名认证、人脸识别 | 必须原件，需在有效期内 |
| **营业执照** | 企业认证、核验经营资质 | 原件/扫描件/原件照片均可 |
| **法人实名银行卡** | 身份验证、开户 | 需与法人身份信息一致 |
| **法人实名手机号** | 接收验证码、账户安全 | 需与税局登记信息一致 |

> **注意**：微业贷申请时仅需线上授权获取信息，无需上传纸质材料。但客户经理在沟通中需确认客户具备以上基本材料。

---

## 二、税务与财务材料

| 材料 | 用途 | 替代材料/说明 |
|------|------|---------------|
| **电子税务局账号密码** | 授权查询纳税记录 | 必须，用于税务认证 |
| **纳税证明** | 核实纳税真实性 | 电子税务局可下载完税证明 |
| **开票记录** | 验证经营规模 | 税控系统/电子税务局导出 |
| **近1年增值税申报表** | 评估经营稳定性 | 电子税务局截图 |
| **企业所得税申报表** | 核实盈利能力 | 季度/年度申报表 |

### 常见问题
**Q：客户没有纳税能申请吗？**
> 微业贷是纳税贷产品，**没有纳税记录的个体户/企业无法申请**。建议：
> - 先确认是否实际有纳税但客户不知道（很多小规模纳税人季度申报）
> - 如确实无纳税，可引导了解其他产品（如微众易贷）
> - 建议客户先规范经营、按时纳税，3-6个月后再申请

**Q：零申报多怎么办？**
> 连续零申报超过9-10个月会影响审批。需了解：
> - 是季节性经营还是实际未经营？
> - 是否有其他收入来源（如个人流水、微信/支付宝收款）？

---

## 三、经营流水材料

| 流水类型 | 证明力 | 适用场景 |
|----------|--------|----------|
| **对公账户银行流水** | ★★★★★ | 最权威，优先采信 |
| **个人银行卡流水（法人）** | ★★★★ | 个体工商户常用，需与经营相关 |
| **微信/支付宝流水** | ★★★ | 补充材料，需完整截图/导出 |
| **POS机流水** | ★★★ | 餐饮、零售等行业可用 |

### 客户经理追问要点
- "您公司主要用哪个账户收款？对公还是个人卡？"
- "月平均流水大概多少？旺季和淡季差别大吗？"
- "流水能覆盖每月还款吗？大概几倍？"
- "微信/支付宝收款占比多少？能导出账单吗？"

---

## 四、经营场所与合同材料

| 材料 | 用途 | 替代方案 |
|------|------|----------|
| **经营场所证明** | 核实真实经营 | 租赁合同、产权证、场地使用证明 |
| **租赁合同** | 验证经营稳定性 | 房东证明+租金转账记录 |
| **采购合同** | 验证进货/备货需求 | 订单、报价单+聊天记录+转账凭证 |
| **销售合同/订单** | 验证下游需求 | 客户订单、框架协议、意向书 |
| **发票** | 验证交易真实性 | 增值税发票、普通发票 |
| **报价单** | 辅助验证采购需求 | 需配合聊天记录或转账凭证 |

### 材料不齐时的替代方案

| 原本需要的材料 | 替代材料组合 |
|----------------|--------------|
| 采购合同 | 报价单 + 供应商聊天记录 + 定金转账记录 |
| 销售合同 | 客户订单截图 + 微信沟通记录 + 历史发货单 |
| 租赁合同 | 房东身份证明 + 租金转账记录 + 现场照片 |
| 纳税证明 | 电子税务局截图 + 税控盘开票记录 |
| 对公流水 | 个人卡流水 + 微信/支付宝账单 + 情况说明 |

> **重要**：替代材料的核心逻辑是**交叉验证**。单一替代材料证明力较弱，需组合使用。

## 五、应收账款与下游材料

| 材料 | 用途 | 风控价值 |
|------|------|----------|
| **应收账款清单** | 评估未来现金流 | 高，但需核实真实性 |
| **下游客户合同** | 验证订单来源 | 中高 |
| **回款证明** | 验证历史回款能力 | 高（银行流水体现） |
| **客户对账单** | 验证交易持续性 | 中 |

### 追问话术
- "您最大的三个客户是谁？合作多久了？账期一般多久？"
- "最近有应收账款没收回来吗？大概多少？"
- "如果这笔贷款下来，您打算怎么还？主要靠哪个客户的回款？"
- "下游客户付款及时吗？有没有拖欠的情况？"

## 六、抵押物/担保材料（如适用）

微业贷为**纯信用贷款**，原则上**不需要抵押物**。

但以下情况可辅助增信：

| 材料 | 作用 | 说明 |
|------|------|------|
| **房产证** | 增信提额 | 不抵押，仅作为资产证明 |
| **车辆登记证** | 增信提额 | 同上 |
| **设备购置发票** | 验证经营投入 | 制造业、加工行业适用 |
| **存货清单** | 辅助评估经营规模 | 批发零售、制造业适用 |

> 部分产品（如微众易贷）明确：名下有房产有机会提升审批通过率和额度。

## 七、贷款用途相关材料

| 用途类型 | 建议准备材料 |
|----------|--------------|
| **采购原材料/进货** | 采购合同/订单、报价单、供应商信息 |
| **支付货款** | 购销合同、发票、对账单 |
| **发放工资** | 员工名单、近3个月工资发放记录 |
| **支付租金** | 租赁合同、租金催缴通知 |
| **设备购置** | 设备报价单、采购合同 |
| **流动资金周转** | 经营流水、订单缺口说明 |

### 禁止用途（客户问起时明确告知）
- ❌ 购买住房及偿还住房抵押贷款
- ❌ 购买理财产品
- ❌ 进入证券市场（股票、债券、期货等）
- ❌ 固定资产、股本权益性投资
- ❌ 偿还其他贷款本金/利息（以贷还贷）
- ❌ 法律法规禁止的其他用途

## 八、材料预审清单（客户经理用）

与客户初次沟通时，可快速确认以下信息：

```
□ 企业名称、统一社会信用代码
□ 法人姓名、身份证号、手机号
□ 营业执照成立日期（是否满18个月）
□ 企业类型（一般纳税/小规模/个体户）
□ 是否有电子税务局账号密码
□ 近一年是否正常纳税（纳税等级）
□ 近一年是否有连续零申报
□ 法人征信是否有当前逾期
□ 近3个月征信查询次数
□ 企业/个人负债大概多少
□ 贷款用途（是否合规）
□ 预计申请金额、期望期限
```

### 预审结论快速判断

| 客户情况 | 建议 |
|----------|------|
| 纳税A/B级 + 征信良好 + 无逾期 | 直接引导申请 |
| 纳税C/M级 + 征信良好 | 可以尝试，额度可能偏低 |
| 无纳税 + 有流水 | 不符合微业贷，推荐其他产品 |
| 有当前逾期 | 建议结清后再申请 |
| 近3个月查询超8次 | 建议养1-3个月征信 |
| 法人超70岁 | 不符合准入 |
| 企业成立未满18个月 | 建议满后再申请 |

## 九、常见客户问题应答

### Q：没有合同怎么办？
> "如果没有正式采购合同，您可以提供报价单加上和供应商的沟通记录、定金转账截图等，这些也可以辅助证明用途。当然，如果后续能补签正式合同会更好。"

### Q：能不能先预审？
> "微业贷是系统实时审批的，您扫码填完信息、完成税务授权后，系统会立刻给出预授信额度，这个过程本身就是最快的'预审'。我可以全程指导您操作，大概5分钟就能看到结果。"

### Q：流水看哪个账户？
> "系统主要抓取的是您企业的纳税和开票数据。如果您有对公账户流水，那是最好的；如果是个体户主要用个人卡收款，也可以作为辅助。关键是有真实经营痕迹。"

### Q：我是空壳公司，能申请吗？
> "微业贷要求有真实经营，系统会核查纳税、开票、工商状态等。如果没有实际经营，建议不要申请，以免浪费征信查询次数。"

### Q：材料我可以慢慢准备吗？
> "微业贷是线上产品，申请时不需要您准备纸质材料，只需要法人身份证、营业执照、电子税务局账号即可。但如果审批通过后需要提款，可能会要求补充用途证明材料。"
