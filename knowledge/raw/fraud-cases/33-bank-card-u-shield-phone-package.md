# 银行卡/U盾/手机号三件套外借

案例类型：账户出借和流水包装

公开来源线索：
反洗钱宣传材料提示，有中介要求客户交出银行卡、U盾、手机号等“三件套”用于包装流水，可能涉及洗钱、电诈资金转移和账户出借风险。

适用场景：客户提到中介要“帮忙养流水”“帮忙走账”，并要求保管银行卡、U盾、手机号或企业网银。

常见表现：
- 中介要求邮寄银行卡、U盾、手机卡或企业网银盾。
- 客户称只是“借账户走一下流水”，不清楚资金来源。
- 账户短期出现多笔陌生资金进出，交易对手分散。
- 资金备注为货款、服务费、备用金，但无真实合同。
- 客户无法控制账户，验证码或网银由他人掌握。

客户经理内部判断：
账户外借可能触发洗钱、帮助信息网络犯罪活动、账户冻结和贷款欺诈风险。若客户已交出账户，应先提示账户安全和报警/挂失风险。

材料核验：
- 账户控制权和网银盾保管人。
- 近期异常流水来源和去向。
- 是否存在账户冻结、司法查询或公安止付。
- 中介聊天记录、邮寄记录、收款承诺。

温和追问：
1. 银行卡、U盾、手机号和企业网银是否都由您本人保管？
2. 是否有第三方帮您走流水或代管账户？
3. 近期账户里陌生资金进出对应什么真实业务？
4. 是否出现过账户被冻结、限额或被要求配合调查？

客户可见边界：
可以明确提醒“银行卡、U盾、手机号和验证码不能外借，账户出借可能带来严重法律风险”。
