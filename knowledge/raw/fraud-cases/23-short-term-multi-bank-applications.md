# 多银行短期申贷

案例类型：多头融资和共债风险

公开来源线索：
经营贷乱象报道提到，部分涉诈贷款业务存在短期内在多家银行申贷等手法。

适用场景：客户近期急需资金，同步咨询多家银行、助贷平台或中介。

常见表现：
- 征信短期内多次贷款审批查询。
- 客户说“哪家先批用哪家”，但用途材料不完整。
- 多个贷款申请金额相近，均声称用于同一采购或周转。
- 客户隐瞒已经获批或已提款的其他授信。
- 短期新增多笔还款义务，经营回款无法覆盖。

客户经理内部判断：
短期多头申贷可能反映真实资金紧张，也可能是骗贷或债务滚动。需核实总资金缺口、已批额度、提款计划和还款来源。

材料核验：
- 征信查询记录。
- 现有授信、贷款余额、月还款额。
- 各平台申请金额和用途。
- 资金缺口测算和回款计划。

温和追问：
1. 近期是否还向其他银行或平台申请过经营贷款？
2. 已经获批或提款的额度分别是多少？
3. 这些贷款是否都对应同一笔采购或周转需求？
4. 全部贷款叠加后，每月还款由哪些经营回款覆盖？

客户可见边界：
不要说“多头骗贷”。可以说“为了避免还款压力被低估，需要把近期申请和已批授信说明完整”。
