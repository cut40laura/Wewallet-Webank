# 空壳企业申请经营贷

案例类型：经营真实性异常

适用场景：企业有营业执照，但真实经营活动弱，借微业贷线上申请便利性包装成经营贷款。

常见表现：
- 企业成立时间较短，或长期零申报后近期突然活跃。
- 客户无法清楚说明经营地址、员工人数、主要客户、供应商和业务流程。
- 对公流水近期突然增加，但交易对手集中、备注笼统、快进快出。
- 无稳定租赁合同、水电物业记录、社保、工资代发。
- 申请额度与纳税、开票、流水、库存和员工规模明显不匹配。

客户经理内部判断：
营业执照真实不等于经营真实。微业贷场景需确认企业是否有持续经营能力和还款来源。空壳风险通常表现为主体材料齐全但经营证据不足。

材料核验：
- 经营场地租赁合同、门头或场地照片、水电物业记录。
- 近12个月对公流水、纳税、开票。
- 员工社保、工资代发、劳动合同摘要。
- 上下游合同、订单、发票、物流或服务交付记录。

温和追问：
1. 企业现在实际经营地址在哪里？是否方便提供租赁合同或物业缴费记录？
2. 目前团队有多少人？工资是通过银行代发还是其他方式发放？
3. 最近三个月流水增长主要对应哪些订单？
4. 主要客户和供应商各有哪几家，合作多久了？

客户可见边界：
不要说“空壳公司”。可以说“经营贷款审批会看持续经营材料，我们需要补充经营场地、流水、税票和上下游交易记录”。
