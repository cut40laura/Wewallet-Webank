# 借用他人企业主体申请

案例类型：主体和实控人不一致

适用场景：贷款申请由非企业实控人推动，法人或股东可能未充分知情。

常见表现：
- 经办人对企业经营非常熟，但营业执照法人或股东不是本人。
- 法人电话无法接通，或法人回答与经办人不一致。
- 客户解释“亲戚公司”“朋友公司”“我帮他操作”，授权材料缺失。
- 登录设备、手机号、银行卡、常用联系人与法人或企业不匹配。
- 经办人催促快速放款，但回避实控关系和还款责任。

客户经理内部判断：
微业贷申请应确保法人、实控人和经办人授权清晰。借用主体可能引发冒名申请、责任不清、资金挪用和贷后失联风险。

材料核验：
- 法人实名确认和贷款意愿确认。
- 企业授权委托书、股东会或内部授权记录。
- 实控人身份证明、持股或实际经营证明。
- 对公账户归属和资金支付审批流程。

温和追问：
1. 这次贷款申请由法人本人操作，还是由您代为经办？
2. 法人和实际经营负责人分别负责哪些事务？
3. 是否方便提供授权材料，确认法人知情并同意申请？
4. 贷款资金发放后由谁安排支付和还款？

客户可见边界：
不要说“冒名申请”。可以说“为了保护企业和法人权益，贷款申请需要确认授权关系和实际经营责任”。
