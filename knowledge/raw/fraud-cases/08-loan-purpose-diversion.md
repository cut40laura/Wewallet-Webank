# 贷款用途挪用

案例类型：资金用途偏离

适用场景：客户申请经营贷，但真实资金用途可能是个人消费、投资、还债或非经营支出。

常见表现：
- 客户用途说法反复变化，从采购、周转改为还款、装修、投资。
- 支付对象不是供应商对公账户，而是个人、亲友、理财平台或陌生公司。
- 客户强调“先放到个人卡再安排”，无法提供具体采购付款计划。
- 近期信用卡、网贷、民间借贷压力明显，贷款可能用于借新还旧。
- 放款后短期内拆分转出，缺少合同、发票、收货凭证。

客户经理内部判断：
微业贷属于企业流动资金贷款服务，资金用途应匹配真实经营。用途挪用会影响还款来源和合规性。需提前确认支付对象、业务合同和贷后凭证。

材料核验：
- 资金支付计划和收款方信息。
- 采购合同、订单、发票。
- 贷后支付凭证、收货或入库记录。
- 征信、流水中近期债务扣款和大额个人支出。

温和追问：
1. 放款后预计支付给哪些供应商？每笔金额大概多少？
2. 收款账户是供应商对公账户吗？合同主体是否一致？
3. 付款后能否补充发票、收货单或入库记录？
4. 除这笔采购外，近期是否还有其他短期还款压力？

客户可见边界：
不要说“套现”或“挪用”。可以说“经营贷需要资金用途和经营交易一致，建议提前准备支付计划和后续凭证”。
