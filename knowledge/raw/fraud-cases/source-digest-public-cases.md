# 公开来源摘要：小微经营贷和微业贷相关欺诈线索

用途：为欺诈案例卡片提供公开来源线索，便于后续复核。以下内容为公开报道和官方页面的摘要，不代表微众银行微业贷真实客诉或内部案件。

## 微业贷官方服务与反诈边界

来源：
- 微众银行微业贷官方页面：https://sme.webank.com/esp-web/rls/asset/wyd/index.html
- 新浪转载微业贷客户经理核验功能报道：https://news.sina.cn/sx/2022-11-24/detail-imqqsmrp7400026.d.html

摘要：
- 微业贷是面向小微企业的线上无抵押企业流动资金贷款产品，主要通过互联网方式服务。
- 官方申请渠道包括“微众银行企业金融”“微众银行微业贷”微信公众号和“微众银行企业金融”App。
- 微业贷提供客户经理身份和来电号码核验能力，用于识别伪冒客户经理。
- 申请微业贷过程中，微众银行不会以任何形式向客户收取中介费、手续费、服务费。

可转化案例：
- 伪冒微众银行人员收费。
- 假客户经理来电。
- 伪造审批截图、提额、解冻、续贷收费。
- 验证码和账号被中介代操作。

## 伪冒涉贷诈骗和小微企业主受害

来源：
- 中国经济网，微众银行持续推进打击金融黑灰产行动，伪冒涉贷诈骗系列案件集中宣判：https://finance.ce.cn/bank12/scroll/202605/t20260507_2950089.shtml
- 央广网，山西警方打掉5个虚假贷款诈骗团伙，380余名小微企业主受害：https://news.cnr.cn/native/gd/20230523/t20230523_526261729.shtml

摘要：
- 公开报道提到，不法团伙冒充微众银行工作人员，伪造工牌、办公定位、贷款审批成功截图等材料，虚构内部渠道，以提额、解冻、续贷等名义骗取高额渠道服务费。
- 山西警方案例中，犯罪团伙以商务咨询服务公司为外衣，通过企业信息查询类App获取企业人员联系方式，自称获得银行授权或是指定合作担保机构，骗取小微企业主高额居间服务费；报道显示受害小微企业主超过380名。

可转化案例：
- 虚假贷款公司居间诈骗。
- 假授权、假担保机构。
- 冒充银行员工和客户经理。
- 高额服务费、保证金、渠道费。

## 经营贷包装乱象

来源：
- 证券时报网，记者起底经营贷乱象：https://www.stcn.com/article/detail/2620853.html

摘要：
- 报道暗访了信贷中介为无真实经营客户包装经营贷的链条，包括代办营业执照、养流水、找经营实体、处理拍照场地。
- 报道提到部分中介使用虚假工资代发、关联交易转账、多个空壳公司账户多次转账等方式制造经营和资金流水假象。
- 报道还提到部分经营贷资金实际流入楼市、金融市场、民间借贷或小贷公司，与经营用途无关。

可转化案例：
- 经营贷“三件套”包装。
- 假场地和挂靠地址。
- 假工资和社保包装。
- 资金快进快出和多空壳账户绕转。
- 经营贷流入房产、理财、股市、民间借贷。

## 经营贷用途挪用和司法后果

来源：
- 新浪财经转载市场资讯，180万经营贷被挪用于还房贷，法院判了：https://finance.sina.com.cn/stock/zqgd/2023-03-02/doc-imyimwvv7840750.shtml

摘要：
- 公开案例中，借款合同约定贷款用于生产经营周转，但借款人被查明将部分经营贷款挪用于偿还房贷，并拒绝提供资金用途证明。
- 法院支持银行提前收回贷款，并确认抵押房产优先受偿等责任。

可转化案例：
- 过桥转贷还房贷。
- 用途证明拒不提供。
- 贷后资金流向无法说明。

## 虚构经营材料和合同骗贷

来源：
- 盛银消费金融消保课堂，缪某虚构经营材料实施贷款诈骗案：https://www.syxfjr.cn/info/xbkt/645
- 中国经济网，商人靠一份假合同骗贷500万：https://www.ce.cn/xwzx/fazhi/201902/18/t20190218_31495468.shtml
- 每日经济新闻，虚增应收账款骗取银行授信和贷款：https://www.nbd.com.cn/rss/toutiao/articles/1331938.html
- 南方+，韶关农行识别虚假银行流水堵截骗贷30万元：https://static.nfapp.southcn.com/content/201703/23/c334332.html

摘要：
- 公开案例涉及设立空壳公司、伪造贸易合同、虚构经营能力、以虚假经营材料骗取无抵押信用贷款。
- 公开案例涉及虚构购买汽车备件等经营行为，提供虚假购销合同取得贷款。
- 公开案例涉及虚构工程或劳务合同、虚增应收账款、虚报经营状况以获取授信或贷款。
- 公开案例也显示银行可通过流水细节识别虚假银行流水。

可转化案例：
- 虚构购销合同骗贷。
- 虚增应收账款。
- 伪造银行流水。
- 空壳企业和虚假经营能力包装。

## 微信公众号和小程序贷款乱象

来源：
- 界面新闻，微信专项处理违规贷款服务，已累计处罚3万余公众号、2千余小程序：https://m.jiemian.com/article/3317706.html
- 东方财经网转载微业贷官方入口反诈提示：https://eastfi.com/m/view.php?aid=19299
- 江西银行反洗钱宣传，警惕流水包装陷阱：https://www.bojx.com/bojx/jyzt/fxqzs/2024062016462029012/index.html

摘要：
- 微信公众平台曾针对虚假、无资质信贷类账号进行专项处理，公开报道提到违规贷款账号常以门槛低、下款快、额度高等噱头诱导用户。
- 微业贷官方入口提示强调，应认准“微众银行企业金融”“微众银行微业贷”等官方微信公众号和官方 App，遇到提现费、中介费、包装费、工本费、押金、保证金、手续费，或要求透露账号密码、验证码的情形，应视为非官方欺诈风险。
- 反洗钱宣传材料提示，有不法中介要求客户交出银行卡、U盾、手机号等“三件套”用于包装流水，可能引发洗钱和账户出借风险。

可转化案例：
- 仿冒官方公众号入口。
- 无资质信贷账号导流。
- 账号、验证码和三件套外借。
- 低门槛、快下款、高额度贷款广告诱导。

## 包装贷和职业背债

来源：
- 新浪财经转载中国经营网，央视曝光包装贷乱象：https://finance.sina.cn/2025-07-22/detail-infhiywn2057262.d.html?vt=4

摘要：
- 公开报道提到，包装贷链条可能通过伪造工作证明、银行流水、收入流水、装修场景、结清证明等方式，将不符合条件的借款人包装成优质客户。
- 报道提到贷款发放后，资金可能通过虚假交易、关联账户层层转移，甚至流向虚拟货币、虚假理财或用于以贷养贷。

可转化案例：
- 征信白户包装成优质客户。
- 虚假场景和虚假结清证明。
- 虚拟货币和提现转移资金。
- 职业背债人和资金被实际控制人转走。
