# 微业贷欺诈案例知识库索引

适用场景：客户经理需要快速匹配异常模式，并生成温和追问。

使用方式：
1. 先根据客户当前异常选择最接近的案例。
2. 只把案例中的“温和追问”和“材料核验”转化为客户可见话术。
3. 不向客户展示案例名称、内部判断或欺诈标签。

案例清单：

| 编号 | 案例类型 | 关键异常 | 推荐动作 |
|------|----------|----------|----------|
| 01 | 虚构采购用途并临时补报价单 | 用途变化、报价单孤证、供应商模糊 | 补合同、订单、历史付款、发票 |
| 02 | 空壳企业申请经营贷 | 无真实经营场地、流水突增、员工缺失 | 核实经营场地、租赁、水电、社保 |
| 03 | 借用他人企业主体申请 | 经办人非实控人、法人不知情 | 核实授权、实控关系、经营分工 |
| 04 | 税票和收入包装 | 近期开票异常、流水与税票不匹配 | 核实发票、合同、物流、回款 |
| 05 | 流水循环做大规模 | 多账户快进快出、交易对手集中 | 核实真实交易和资金闭环 |
| 06 | 虚构下游订单 | 大额订单无历史合作、回款周期不合理 | 核实下游合同、对账、收货 |
| 07 | 中介包装收费诈骗 | 承诺包过、收保证金或服务费 | 强化官方渠道和免中介费提示 |
| 08 | 贷款用途挪用 | 经营贷转消费、投资、还债 | 核实支付对象和贷后凭证 |
| 09 | 身份与控制权不一致 | 证照真实但操作人异常 | 核实设备、手机号、法人意愿 |
| 10 | 多平台叠加负债 | 短期多头申请、隐瞒债务 | 核实征信、流水扣款和负债 |
| 11 | 伪冒微众银行人员收费 | 假工牌、假截图、称有内部渠道 | 引导客户核验身份，提示不收费 |
| 12 | 虚假贷款公司居间诈骗 | 自称银行授权，骗取高额居间费 | 核实授权来源和收费情况 |
| 13 | 经营贷“三件套”包装 | 代办执照、养流水、找假实体 | 核实经营连续性和场地真实性 |
| 14 | 假场地和挂靠地址 | 拍照场地非实际经营地 | 核实租赁、水电、访谈和轨迹 |

| 16 | 过桥转贷还房贷 | 经营贷置换房贷或过桥资金 | 核实提前还贷资金来源 |
| 17 | 经营贷流入投资理财 | 贷款转股市、基金、理财 | 核实贷后资金路径 |
| 18 | 假审批截图和解冻费 | 伪造审批成功、收解冻费 | 反诈提示并核验官方页面 |
| 19 | 验证码和账号代操作 | 中介掌握客户登录和验证码 | 核实本人操作和账户控制权 |
| 20 | 虚构购销合同骗贷 | 合同真实交易不存在 | 核验交易闭环和履约能力 |
| 21 | 虚增应收账款 | 伪造工程或销售应收 | 核实回款方、验收和账龄 |
| 22 | 伪造银行流水 | 流水截图或纸质流水造假 | 核验原始流水和银行回单 |
| 23 | 多银行短期申贷 | 同期向多家机构申请 | 核实征信查询和资金缺口 |

| 25 | 内外勾结包装进件 | 中介熟悉流程、疑似内部配合 | 升级复核，留存异常证据 |

| 28 | 关联企业转手贸易 | 多家关联公司制造贸易链 | 穿透关联关系和最终收款方 |
| 29 | 供应链核心企业名义冒用 | 借大客户名义虚构订单 | 核实核心企业确认和对账 |
| 30 | 贷后凭证补造 | 放款后补假发票、假物流 | 要求原始凭证和第三方核验 |
| 31 | 仿冒官方公众号入口 | 假公众号、假小程序、假链接 | 核实官方入口和认证主体 |
| 32 | 无资质信贷账号导流 | 低门槛、快下款、高额度诱导 | 核实牌照、主体和收费 |
| 33 | 银行卡/U盾/手机号三件套外借 | 出借账户包装流水或洗钱 | 立即提示账户安全和法律风险 |
| 34 | 征信白户包装成优质客户 | 伪造工作、收入、资产 | 核实资信来源和资金去向 |
| 35 | 虚拟货币和提现转移资金 | 放款后通过隐蔽渠道转移 | 核实最终收款方和资金闭环 |

统一客户可见边界：
贷款材料不是越多越好，而是要能互相印证。对客户只强调“真实经营、真实用途、真实还款来源”，不讨论内部欺诈模型。
