# 征信白户包装成优质客户

案例类型：资信材料包装

公开来源线索：
公开包装贷报道提到，不法中介会通过伪造工作证明、工资流水、收入流水、资产交易等方式，将不符合条件的借款人包装成优质客户。

适用场景：客户征信记录薄、经营历史短，但短期内出现稳定工资、资产、流水和经营证明。

常见表现：
- 客户长期无稳定收入记录，近期突然出现高额工资或经营流水。
- 工作证明、收入证明、社保、个税和银行流水互相不匹配。
- 客户不熟悉“任职公司”或交易对手。
- 资产交易价格异常，可能用于抬高资质。
- 中介承诺“白户也能包装成优质客户”。

客户经理内部判断：
征信白户不是风险本身，风险在于资信证明是否真实、连续、可核验。应核验收入来源、经营能力和实际还款来源。

材料核验：
- 工资、社保、个税、劳动合同一致性。
- 收入流水形成时间和来源。
- 资产交易合同、付款路径和市场价格。
- 客户对工作、经营和交易细节的熟悉程度。

温和追问：
1. 您目前主要收入来自工资、经营收入还是其他来源？
2. 近期收入增长对应哪些真实业务或岗位变化？
3. 工作证明、社保和工资代发是否能相互对应？
4. 是否有第三方帮您整理或制作资信材料？

客户可见边界：
不要说“白户包装”。可以说“资信材料需要真实、连续、可核验，短期新增材料需要解释形成原因”。
