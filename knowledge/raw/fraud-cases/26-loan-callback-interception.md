# 贷款回访拦截

案例类型：客户真实意愿异常

公开来源线索：
虚假贷款中介和经营贷包装链条常要求客户按指定话术回答问题，甚至代接回访或实时提示。

适用场景：客户回答高度模板化，或回访过程中有第三方干预。

常见表现：
- 客户回答像背稿，停顿后才回答关键问题。
- 背景有人提示，或客户多次说“我问一下”。
- 客户对贷款用途、供应商、合同、还款来源细节不熟。
- 回访号码不是法人或实控人常用号码。
- 回访后客户又通过中介补充“标准答案”。

客户经理内部判断：
回访真实性关系到本人意愿、用途真实性和授权有效性。应通过开放式问题确认客户是否独立理解并承担责任。

材料核验：
- 常用联系方式和实名归属。
- 法人、实控人、财务分别确认。
- 申请资料和客户口述一致性。
- 第三方代办、代接或远程指导记录。

温和追问：
1. 这笔贷款用途请您用自己的话再说明一下。
2. 供应商和回款客户分别是谁，您平时和谁对接？
3. 现在是否方便您本人独立确认，不需要第三方代答？
4. 贷款合同责任和还款安排您是否都清楚？

客户可见边界：
不要说“你在背话术”。可以说“回访需要确认您本人真实意愿，建议由法人或实际经营人独立完成”。
