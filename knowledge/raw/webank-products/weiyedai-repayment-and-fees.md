## 一、计息规则

### 核心原则：按日计息，借几天算几天

微业贷采用**按日计息**方式，利息仅按实际借款天数计算，不使用不收费，提前还款无违约金。

### 利息计算公式

```
每日利息 = 剩余未还本金 × 日利率
某期利息 = 借款本金 × 日利率 × 当期实际计息天数
```

其中：
- **日利率** = 年化利率 ÷ 365（以页面展示为准）
- **计息天数**：算头不算尾，即从放款当日开始计息，还款日当天不计息

### 示例计算

**示例1：短期周转**
- 借款金额：50万元
- 年化利率：6%（日利率约0.0164%）
- 借款期限：30天
- 利息 = 500,000 × 6% ÷ 365 × 30 ≈ 2,465.75元

**示例2：等额本金还款（首月）**
- 借款金额：100万元
- 年化利率：8%（日利率约0.0219%）
- 期限：12期
- 首月本金 = 1,000,000 ÷ 12 = 83,333.33元
- 首月利息 = 1,000,000 × 8% ÷ 12 ≈ 6,666.67元
- 首月还款 = 83,333.33 + 6,666.67 = 90,000元

> **客户可见话术**："微业贷是按日计息的，您借多少天就算多少天利息，比如日利率是万分之二，借10万块钱，一天利息就是20块。提前还款也没有违约金，资金宽裕了随时可以申请提前还。"

## 二、还款方式详解

### 1. 等额本金

- **特点**：每月偿还同等本金，利息随本金减少而逐月递减
- **月供趋势**：逐月递减
- **总利息**：相对较少
- **适合客户**：前期还款能力较强、希望总利息支出较少的客户

| 期数 | 月供本金 | 月供利息 | 月供合计 |
|------|----------|----------|----------|
| 第1期 | 83,333 | 6,667 | 90,000 |
| 第2期 | 83,333 | 6,111 | 89,444 |
| ... | ... | ... | ... |
| 第12期 | 83,333 | 556 | 83,889 |

### 2. 等额本息

- **特点**：每月偿还同等金额（本金+利息）
- **月供趋势**：固定不变
- **总利息**：相对较多
- **适合客户**：希望月供稳定、便于财务规划的客户

### 3. 先息后本（限优质客户）

- **特点**：前3期或前6期仅偿还利息，不还本金；后续转为等额本金/本息还款
- **前期压力**：极低
- **总利息**：相对较多
- **适合客户**：前期资金紧张、但预期后期有回款或收入增长的客户

> **客户经理提示**：并非所有客户都能看到全部还款方式，系统会根据客户资质展示可选方案。客户咨询时应说明"以您申请时页面展示的还款方式为准"。

## 三、还款操作指南

### 自动还款（推荐）

1. 在借款时或还款日前，确保绑定的还款银行卡内余额充足
2. 系统在还款日凌晨自动扣款
3. 扣款成功后，客户会收到短信通知
4. 如扣款失败，系统会在还款日期间多次尝试

### 手动还款

1. 打开"微众银行企业金融"App或公众号
2. 进入"我的贷款"或"还款"页面
3. 选择当期应还或提前结清
4. 确认还款金额，选择还款银行卡
5. 输入交易密码完成还款

### 提前还款

#### 操作步骤

1. 进入App/公众号"我的贷款"页面
2. 点击"提前还清借款"或"提前还款"
3. 选择要提前结清的借款笔数
4. 系统自动核算应还总金额（剩余本金 + 截至还款日利息）
5. 确认后通过绑定银行卡完成还款

#### 提前还款规则

| 项目 | 规则 |
|------|------|
| **违约金** | 无 |
| **手续费** | 无 |
| **最低时限** | 借款次日即可申请提前还款 |
| **计息截止** | 计算到提前还款日前一天，还款日当天不计息 |
| **还款类型** | 需一次性结清整笔借款，不支持部分提前还款 |
| **额度恢复** | 还款成功后额度实时恢复，可再次支用 |

> **重要提示**：任何以"提前还款手续费""保证金""解冻费"等名义要求缴费的行为，均为诈骗。微业贷提前还款不收取任何费用。

## 四、费用总览

### 微业贷费用清单

| 费用项目 | 是否收取 | 金额 |
|----------|----------|------|
| **贷款利息** | 是 | 按日计息，以页面展示利率为准 |
| **授信评估费** | 否 | 免费 |
| **账户管理费** | 否 | 免费 |
| **提款手续费** | 否 | 免费 |
| **提前还款违约金** | 否 | 免费 |
| **中介费/服务费** | 否 | 官方不收取任何形式的额外费用 |

### 逾期费用

如客户未在还款日足额还款，将产生以下后果：

1. **逾期利息**：按合同约定利率计收逾期利息
2. **征信影响**：逾期记录将上报企业征信和个人征信
3. **额度冻结**：严重逾期可能导致额度冻结或提前收回贷款
4. **催收程序**：银行将通过短信、电话等方式提醒还款，必要时采取法律手段

> **合规提示**：客户经理应引导客户按时还款，维护良好征信记录。不可帮助客户隐瞒逾期或协商"内部消逾期"。

## 六、还款常见问题

### Q1：还款日可以修改吗？

还款日由系统在放款时自动确定，通常根据放款日期对月对日计算。**不支持人工修改还款日**。

### Q2：还款银行卡可以更换吗？

可以。客户可在App/公众号的"还款设置"中更换绑定银行卡，新卡需为法人本人名下的借记卡。

### Q3：还款日遇到节假日怎么办？

如还款日为法定节假日，通常不顺延，系统仍会在当日尝试扣款。建议客户提前1-2天确保账户余额充足。

### Q4：逾期后还能继续借款吗？

逾期状态下通常无法发起新的借款申请。需结清逾期款项后，系统重新评估方可恢复借款功能。

### Q5：如何获取还款凭证/结清证明？

客户可在App/公众号的"贷款记录"中下载电子还款计划表和结清证明。如需纸质材料，可联系官方客服95384。

## 七、额度不用是否收费？

**不收费**。微业贷仅在资金实际使用后开始计息：

- 授信额度获批后，不使用不产生任何费用
- 额度有效期内可随时提款
- 用款时按实际借款金额和天数计息
- 额度到期后如未使用，自动失效，不影响客户征信

> **客户可见话术**："额度批下来之后，您不用的话是不收任何费用的，只有真正借出来用了才开始算利息。您可以先把额度批下来，需要周转的时候随时提。"
