# 微业贷准入要求与禁入规则

资料来源：
- 微众银行官方产品页及合作渠道公开资料
- 渣打银行微业贷银银合作业务说明
- 行业公开信息整理（51卡农社区、我要有钱网等）
- 信息整理日期：2026-05-21

---

## 一、申请人（法定代表人）要求

### 基本准入

| 项目 | 要求 | 备注 |
|------|------|------|
| **身份** | 企业法定代表人 | 必须为工商登记在册的法人 |
| **年龄** | 18～70周岁 | 部分渠道显示20～60周岁为更优区间 |
| **持股要求** | 无明确占股要求 | 即使不占股也可申请 |
| **变更要求** | 无硬性时间限制 | 法人变更后，工商和税务系统同步即可申请 |

### 征信要求

微业贷同时参考企业征信和法人个人征信，以下为行业总结的征信红线：

1. **当前逾期状态**
   - 当前无逾期记录
   - 贷记卡、准贷记卡、贷款、对外担保的五级分类状态不可为：次级、可疑、关注、损失
   - 不可出现担保人代偿、以资抵债等异常状态

2. **历史逾期要求**
   - 24个月内无"4"（连续逾期4个月）
   - 24个月内无累计7个"1"
   - 12个月内无"3"（连续逾期3个月）
   - 6个月内无"2"（连续逾期2个月）
   - 近2年逾期或欠息在30天以内的次数建议低于6次

3. **查询次数建议**
   - 无明确硬性门槛
   - 建议：2个月内贷款审批查询≤6次，3个月内≤8次
   - 频繁查询可能触发风控降额或拒贷

4. **其他征信状态**
   - 贷记卡、准贷记卡不可为冻结、呆账、止付状态
   - 房屋抵押贷款记录为加分项
   - 设备租赁贷款和小额贷款记录可能为减分项

5. **纯白户**
   - 纯白户（无征信记录）存在批款案例，但通过率相对较低，需谨慎评估

## 二、企业准入要求

### 企业基本信息

| 项目 | 要求 |
|------|------|
| **成立时间** | 营业执照满18个月（部分渠道显示满2年） |
| **纳税时间** | 正常纳税满18个月 |
| **企业类型** | 一般纳税人、小规模纳税人、个体工商户均可 |
| **禁入类型** | 分公司、外资企业、港台合资企业、合伙企业、合作社 |

### 纳税要求

纳税记录是微业贷审批的核心依据之一，具体要求如下：

1. **纳税等级**
   - 准入：A、B、C、M级或无评级
   - 禁入：D级（部分渠道反馈D级也有极小概率批款）
   - 注意：浙江地区C级可能暂不准入

2. **纳税金额**
   - 近2年内至少有一年的增值税纳税额不为0
   - 无连续10个月以上的0申报记录
   - 近一年连续缴纳增值税1万元以上（供应链/基建贷等产品要求）

3. **申报状态**
   - 近1年增值税、企业所得税申报正常
   - 不存在逾期未缴款情况
   - 近半年历史滞纳金次数≤3次

4. **开票记录**
   - 有正常经营开票记录为佳
   - 近12个月有效销项发票张数建议≥30张
   - 红票占比建议控制在合理范围

### 司法与诉讼要求

- 公司及法人无未结刑事诉讼
- 对诉讼案件持开放态度，但非当前被执行案件为最佳
- 当前无涉诉或已结案为佳
- 非失信被执行人、非限高客户

## 三、禁入行业

以下行业的客户无法申请微业贷（或通过率极低）：

| 类别 | 具体行业 |
|------|----------|
| **金融业** | 银行、证券、保险、投资、担保、P2P、小额贷款、典当 |
| **房地产业** | 房地产开发、房地产中介、房地产租赁经营 |
| **公共管理与社会组织** | 公共管理、社会保障、社会组织、国际组织 |
| **高污染/高能耗** | 煤炭开采、造纸、化工制造 |
| **其他敏感行业** | 娱乐会所、危化品、法律服务（部分渠道限制） |

### 偏好行业

以下行业在审批中相对更受青睐：

- 批发业和零售业
- 制造业
- 建筑业（可申请专门的"基建贷"产品）
- 租赁和商务服务业
- 科学研究与技术服务业
- 信息技术服务业
- 科创型企业（可申请"科创贷款"专项产品）

> **客户经理提示**：客户询问自身行业是否准入时，应说明"具体以系统审批结果为准，微业贷覆盖绝大多数实体经营行业，您可以先尝试申请查看额度"。

## 五、银银合作特殊准入

微业贷与渣打银行、星展银行等外资银行开展银银合作业务，合作产品有以下特殊规则：

### 渣打银行合作模式

- **出资比例**：渣打银行70%，微众银行30%
- **客户群体**：中国境内注册满一年的非外商投资/非外商独资小微企业
- **担保要求**：个人担保人满21周岁到70周岁
- **额度上限**：每笔总授信额度不超过人民币300万元
- **合同签署**：分别与渣打银行和微众银行签订贷款合同
- **利率展示**：支用阶段平台会明确展示贷款银行为渣打银行

### 星展银行合作模式

- 星展银行出资比例70%，微众银行30%
- 主要用于"微粒贷"个人消费贷合作，微业贷领域亦有类似合作架构

> **客户经理提示**：银银合作业务的额度、利率、还款方式以合作银行及微众银行共同确定的页面展示为准。客户在提款时应注意识别放款银行信息。

## 六、常见被拒原因与再申请建议

### 常见被拒原因

1. 企业成立时间不足或纳税记录不符合要求
2. 纳税等级为D级或存在严重欠税记录
3. 法人或企业当前有逾期、被执行、失信记录
4. 征信查询次数过多，触发风控规则
5. 所属行业为禁入行业
6. 企业经营数据（流水、开票）与申请额度不匹配
7. 近期有频繁申请贷款记录

### 再申请建议

- 如被拒，建议先改善经营数据和征信状况
- 一般建议间隔3个月以上再尝试申请
- 可关注官方"贷中提额"或重新授信活动
- 如企业资质有显著改善（如纳税等级提升、开票增长），可随时尝试

> **合规话术**："微业贷的审批由系统全自动完成，结果不受人工干预。如果这次没有通过，建议您关注企业的纳税记录和征信状况，过段时间可以再尝试。"
