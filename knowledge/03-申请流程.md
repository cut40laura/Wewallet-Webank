# 第三优先级：申请流程

> **用途**：客户进入行动阶段后，Agent 能自然推进，而不是一直风控追问。
> **更新日期**：2026-05-20

---

## 一、线上申请总体流程

```
【客户准备】
    ↓
[1] 扫码/关注公众号/打开APP
    ↓
[2] 个人实名认证（法人）
    ↓
[3] 企业认证（税务认证/工商认证）
    ↓
[4] 授权查询征信、纳税数据
    ↓
[5] 系统审批，获取额度
    ↓
[6] 视频核验（部分客户需要）
    ↓
[7] 签约开户（如未开户）
    ↓
[8] 发起借款，填写收款账户
    ↓
[9] 提款到账（最快1分钟）
    ↓
[10] 按约还款
```

> **全流程线上完成，无需线下开户，无需提交纸质材料。**

---

## 二、详细步骤拆解

### 步骤1：进入申请入口

| 方式 | 操作 |
|------|------|
| **微信公众号** | 微信搜索"微众银行企业金融" → 关注 → 菜单栏"贷款"或"申请额度" |
| **APP** | 下载"微众银行企业金融"APP → 注册/登录 |
| **小程序** | 微信小程序搜索"微众银行企业金融" |
| **扫码** | 扫描官方推广二维码 |

### 步骤2：个人实名认证（法人）

需要准备：
- 法人身份证原件
- 法人实名银行卡
- 法人实名手机号（接收验证码）

认证内容：
1. 上传身份证正反面照片
2. 绑定实名银行卡
3. 人脸识别/活体检测
4. 设置交易密码
5. 签署相关授权协议

> **常见问题**：提示"非法人"
> - 核实手机号、银行卡号、税局登记信息是否实名一致
> - 更换实名银行卡认证
> - 下载"微众银行企业金融"APP或"微众银行"APP重新做实名认证
> - 如仍报错，说明法人在禁入名单

### 步骤3：企业认证

企业认证有两种方式：

| 方式 | 适用地区 | 操作步骤 |
|------|----------|----------|
| **税务认证** | 支持地区 | 跳转至所在地税局页面 → 输入电子税务局账号密码 → 完成授权 |
| **工商认证** | 不支持税务认证地区 | 上传营业执照照片 → 系统自动识别企业信息 → 法人确认 |

**需要提前授权的地区**（以最新系统为准）：
- 北京、辽宁（大连）、山东（除青岛）、河北、江西、四川、湖北
- 安徽（自然人授权）、重庆、广西、福建（自然人授权）

> 授权方式：登录账号为法人身份证号或手机号，密码为税局登录密码。

### 步骤4：授信申请与审批

完成认证后：
1. 系统自动拉取企业纳税、开票、征信、工商等数据
2. 大数据风控模型实时评估
3. **最快1分钟出额度**
4. 页面展示：授信额度、利率、期限

> 审批结果不受人为干预，系统全自动处理。

### 步骤5：视频核验（部分客户）

触发场景：
- 系统风控要求补充核验
- 大额提款前核验
- 信息变更后核验

视频核验要求：
1. 需本人脸部识别
2. 实时拍摄，不能用照片
3. 拍摄清晰，与身份证照片匹配
4. 正对手机，拍下整个面部
5. 光线充足、网络稳定
6. 普通话标准
7. 确保无其他人影/声音进入

> 人脸验证失败处理：
> - 核实是否正在办理户口迁移或身份证更新
> - 关机重启或重装APP
> - 换亲友手机尝试

### 步骤6：签约开户

如客户未在微众银行开户：
1. 系统自动引导开户
2. 填写企业基本信息
3. 上传营业执照
4. 填写基本存款账户编号
5. 完成开户

> 开户失败处理：
> - 在线更换其他实名银行卡
> - 下载"微众银行"APP完成实名认证后再操作
> - 检查基本账户编号是否输入正确

### 步骤7：提款操作

1. 在额度页面点击"借款"
2. 选择借款金额、期限
3. 填写收款账户（**必须是对公账户**，不能是个人卡）
4. 确认利率、还款计划
5. 法人刷脸验证
6. 签署电子合同
7. 等待放款

> **提款注意事项**：
> - 收款账户必须是**企业对公账户**
> - 如找不到开户行，建议开立微众银行单位结算户
> - 建议工作日9:00-17:00发起，100万以上大额非工作日可能因人行大额支付渠道关闭而失败
> - 每一笔借款需法人自主电签，系统自动审核

### 步骤8：还款

#### 正常还款
1. 登录"微众银行企业金融"公众号
2. 点击"查询" → "还款查询"
3. 查看当前应还金额、还款时间
4. 查询专用还款账户
5. 从企业他行对公账户转账至微众银行专用还款账户
6. 系统在账单日自动批处理扣款

> 转账信息填写示例：
> - 收款银行：微众银行/深圳前海微众银行股份有限公司
> - 收款账号：系统分配的专用还款账户
> - 收款户名：XXXX有限公司企业贷还款:微业贷还款
> - 金额：系统提示的应还金额
> - 转账方式：实时到账

#### 提前还款
- 支持**提前全部结清**
- **不支持部分提前还款**
- 借款当天不可提前还款
- 提前还款**不收取手续费**
- 部分优惠套餐（如30万活动）可能有90天内不可提前还款限制

> 提前结清流程：登录APP/公众号 → 选择借据 → 提前结清 → 确认还款金额 → 转账还款

## 三、审批与放款时效

| 环节 | 时效 |
|------|------|
| 系统审批出额 | 最快1分钟 |
| 视频核验（如需） | 实时完成 |
| 开户（如需） | 即时 |
| 放款到账 | 最快1分钟，一般30分钟内 |
| 整体时效 | 资质好的客户从申请到放款可当天完成 |

## 四、借款后管理

### 4.1 续贷
- 正常还款客户，额度到期后可重新申请
- 系统根据历史还款表现、最新经营数据重新评估
- 无固定续贷时间，以系统评估为准

### 4.2 提额
- 保持良好还款记录
- 补充纳税/开票数据（额度可能自动提升）
- 部分产品支持"额度加油站"功能，补充资产证明提额
- 提额幅度因人而异，以系统审批为准

### 4.3 额度冻结/失效
可能原因：
- 法定代表人信息变更
- 人脸验证失败
- 企业/法人信息与系统不一致
- 风险预警触发
- 长期未使用（部分额度可能收回）

恢复方式：
- 更新认证信息
- 联系客服95384核实原因
- 重新申请评估

## 五、常见失败原因与处理

### 5.1 申请阶段失败

| 失败原因 | 表现 | 处理建议 |
|----------|------|----------|
| **税务授权失败** | 无法跳转税局/授权超时 | 重新授权或等待系统同步 |
| **身份验证失败** | 提示非法人/信息不符 | 核对实名信息一致性；变更后等3个工作日 |
| **企业认证失败** | 无法验证企业身份 | 清理缓存、换网络、换进件码重试 |
| **暂无额度** | 审批未通过 | 养好征信和纳税，1-3个月后重试 |
| **黑名单** | 开户/认证报错 | 暂时不符合，过段时间再尝试 |

### 5.2 提款阶段失败

| 失败原因 | 表现 | 处理建议 |
|----------|------|----------|
| **收款账户异常** | 搜不到银行/账户状态异常 | 开微众单位结算户；完成银企对账 |
| **签约失败** | 一直显示签署中 | 核实是否完成刷脸视频验证 |
| **借款审核未通过** | 提交后拒绝 | 核额后太久未用需重新核定；保持良好征信后再试 |
| **非工作日大额** | 100万以上提款失败 | 工作日9:00-17:00再试 |
| **风险预警冻结** | 借款按钮灰色 | 联系95384询问冻结原因 |
| **信息不全** | 无报错但不到账 | 核实是否变更个人/企业信息，联系95384补全 |

### 5.3 还款阶段问题

| 问题 | 处理 |
|------|------|
| 找不到微众银行 | 搜索"深圳前海微众银行"；或到柜面转账 |
| 转账后未扣款 | 查询还款户余额；系统在账单日自动批扣 |
| 逾期了怎么办 | 立即补足还款；逾期影响征信并产生逾期利息 |
| 想提前还款找不到入口 | APP/公众号 → 借据详情 → 提前结清 |

## 七、客户推进话术（行动阶段）

### 引导申请
> "您条件看起来符合基本要求，我帮您算一下：您企业成立X年，纳税等级X级，征信也正常。现在申请最快1分钟就能看到额度，我指导您操作，大概5分钟搞定。您现在方便吗？"

### 遇到认证问题
> "没关系，这是常见的系统同步问题。我们先确认一下：您的手机号、银行卡、税局登记是不是同一个身份证？我教您一步步排查。"

### 审批未通过
> "系统这次没有通过，可能是XX原因（查询多/负债高/数据不匹配）。建议您接下来X个月：①不要点其他网贷；②按时纳税；③降低负债。过段时间我提醒您再试。"

### 提款指导
> "额度出来了！现在提款需要确认几点：①收款账户是对公账户；②金额和期限按您需求选；③法人需要刷脸确认。您准备好对公账户信息了吗？"
