## 微业贷一句话定位

**微业贷** = 微众银行面向小微企业/个体工商户的 **纯信用、全线上、按日计息、随借随还** 的流动资金贷款。

核心数字记忆：
- 额度：最高 **1000万**
- 利率：**3.6%-18%**（单利）
- 期限：**12/24/36期**
- 速度：最快 **1分钟**到账
- 门槛：**无需抵押、无需担保、无需线下开户**

---

## 关键合规红线（全员必背）

1. **不承诺结果**：不说"保证通过""一定放款""最低利率"
2. **不引导违规**：不帮客户虚构用途、伪造材料
3. **不收费**：不收取任何中介费、手续费、加急费
4. **真实用途**：经营贷必须用于生产经营，严禁购房、炒股、还旧贷
5. **官方渠道**：只引导客户通过公众号/APP/小程序申请
