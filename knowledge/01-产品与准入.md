# 第一优先级：产品与准入

> **用途**：客户问"我能不能贷、能贷多少、多久、利率多少"时用。
> **来源**：微众银行官网、微众银行企业金融官方页面、公开产品说明、政府/税务平台引用页、行业公开资料
> **更新日期**：2026-05-20
> **免责声明**：本知识库基于公开信息整理，具体产品要素以微众银行官方最新审批结果和合同约定为准。

## 一、微业贷产品介绍

### 1.1 产品定义
微业贷是**微众银行**面向**小微企业、个体工商户**等经营性客户推出的**纯信用、线上化**流动资金贷款产品。它是国内首个全线上、纯信用的企业流动资金贷款产品。

### 1.2 核心特点
| 特点 | 说明 |
|------|------|
| **纯信用** | 无需抵押、无需担保、无需线下开户 |
| **全线上** | 从申请到提款全部在线完成，无需纸质材料 |
| **速度快** | 额度立等可见，资金最快1分钟到账 |
| **按日计息** | 用一天算一天，随借随还 |
| **无中介费** | 系统全自动审批，无需向任何人支付中介费 |

### 1.3 产品定位
- 服务客群：年营业收入1000万元以下的小微企业占比约77%
- 首贷户友好：约66%的小微企业在授信时没有任何银行企业贷款记录
- 行业分布：主要集中在**批发零售业、制造业、建筑业、高新科技**

## 二、产品要素（额度/期限/利率/还款方式）

### 2.1 额度范围
| 项目 | 标准版 | 备注 |
|------|--------|------|
| **最高额度** | 1000万元 | 官网展示最高可达1000万 |
| **常见额度** | 5万-500万 | 多数客户获批区间 |
| **户均授信** | 约99万元 | 基于公开业务数据 |
| **笔均借款** | 约20万元 | 实际支用均值 |

> **客户经理提示**：具体额度由系统根据企业纳税/开票、征信、经营流水、工商状态等多维数据自动核定，人工无法干预。不可向客户承诺具体额度。

### 2.2 借款期限
| 期限选项 | 说明 |
|----------|------|
| 12期（1年） | 最常见 |
| 24期（2年） | 中等期限 |
| 36期（3年） | 最长期限 |

> 系统根据客户资质自动判定可贷期限，非客户自选。

### 2.3 利率口径
| 利率类型 | 范围 | 说明 |
|----------|------|------|
| **年化利率（单利）** | **3.6% - 18%** | 主流公开区间 |
| **日利率** | 约0.02% - 0.05% | 按日计息换算 |
| **最低利率** | 3.6%起 | 纳税稳定、信用良好的优质客户 |

> **重要**：
> - 利率为**单利计息**，非复利
> - 具体利率以**审批结果为准**，页面展示为准
> - 新客常有优惠活动（如"新客享借300万免息福利"等，以当时活动为准）
> - **不可承诺"最低利率"或"一定拿3.6%"**

### 2.4 还款方式
| 方式 | 适用场景 |
|------|----------|
| **等额本金** | 最常见默认方式 |
| **先息后本** | 部分客户前3期先息后本+后续等额本金 |
| **随借随还** | 按实际使用天数计息，提前还款无违约金 |
| **等额本息** | 部分产品版本支持 |

> 提前还款：支持提前全部结清，**不支持部分提前还款**。借款当天不可提前还款。提前还款**不收取手续费**（具体以借款合同为准）。

## 三、企业贷 vs 个人经营贷 区别

微众银行体系内，企业经营性贷款与个人经营性贷款的核心区别：

| 维度 | 微业贷（企业贷） | 微众易贷/个人经营贷 |
|------|------------------|---------------------|
| **借款主体** | 企业（有限公司/个体户） | 个人（个体工商户/小微企业主） |
| **征信体现** | 上企业征信 + 法人担保上个人征信 | 上个人征信 |
| **额度上限** | 最高1000万 | 最高50万（微众易贷） |
| **申请条件** | 需企业纳税/开票、营业执照满18个月 | 营业执照满1年即可 |
| **放款账户** | 对公账户 | 个人账户 |
| **利率区间** | 3.6%-18% | 6%-19.8% |
| **担保方式** | 法人不可撤销担保 | 纯信用 |
| **适用场景** | 企业流动资金周转、进货、发工资等 | 小额经营周转 |

### 与微粒贷的区别
| 维度 | 微业贷 | 微粒贷 |
|------|--------|--------|
| **性质** | 经营性贷款 | 个人消费贷款 |
| **用途** | 企业生产经营 | 日常消费 |
| **额度** | 最高1000万 | 最高30万 |
| **客群** | 企业法人/个体户 | 个人消费者 |

## 四、适用对象

### 4.1 可申请的主体类型
1. **小微企业**（有限公司、股份有限公司等）
2. **个体工商户**
3. **企业法定代表人**亲自申请（不接受股东代申请）

### 4.2 申请人（法人）要求
| 条件 | 具体要求 |
|------|----------|
| **年龄** | 18周岁（含）- 70周岁（含） |
| **身份** | 仅支持中国大陆居民身份证 |
| **持股比例** | 不占股也可申请 |
| **变更要求** | 法人变更无特殊时间限制（工商信息同步即可） |

### 4.3 企业要求
| 条件 | 具体要求 |
|------|----------|
| **企业类型** | 一般纳税人、小规模纳税人、个体工商户 |
| **成立时间** | 营业执照满18个月（部分信息源称企业满2年、个体户满1年） |
| **纳税要求** | 连续纳税满18-24个月，年均纳税额1万元以上（增值税+所得税） |
| **纳税等级** | A/B/C/M级均可，无评级有小概率批款 |
| **零申报** | 避免超过9-10个月连续零申报 |
| **涉诉状态** | 当前无重大涉诉、无当前逾期贷款 |

## 六、准入条件详解

### 6.1 经营年限
- 企业成立并实际经营**满18个月以上**
- 个体工商户部分通道要求满1年
- 法人变更后需等待工商信息同步（一般3个工作日左右）

### 6.2 纳税/开票要求
| 指标 | 要求 |
|------|------|
| **纳税时长** | 连续纳税18-24个月以上 |
| **纳税金额** | 年均缴纳增值税+所得税1万元以上 |
| **纳税等级** | A/B/C/M级优先，无评级可尝试 |
| **零申报** | 近2年任一年度增值税纳税额不为0；避免10个月以上连续零申报 |
| **逾期缴款** | 近一年无逾期未缴款 |
| **滞纳金** | 近半年历史滞纳金次数限制在3次以内 |

### 6.3 征信要求（法人+企业）
| 指标 | 要求 |
|------|------|
| **当前逾期** | 企业和法人当前均无逾期 |
| **历史逾期** | 近2年内逾期或欠息30天以内不超过6次；无60天以上逾期 |
| **连三累六** | 近2年无连续3个月逾期或累计6次逾期 |
| **查询次数** | 近3个月征信查询不超过8次（部分口径称4次） |
| **负债率** | 企业负债率建议不超过70%；信用类负债不超过500万（抵押类不算） |
| **账户状态** | 无"冻结""呆账""止付"状态 |
| **五级分类** | 无"次级""可疑""损失" |
| **对外担保** | 对外担保贷款状态不可为次级、可疑、损失 |

### 6.4 工商状态要求
- 企业状态为"在营（开业）"
- 无经营异常名录记录
- 无严重违法失信记录
- 近6个月无法定代表人或股东变更（部分产品要求）

### 6.5 法人身份要求
- 必须是企业法定代表人
- 持有中国大陆居民身份证
- 年龄在18-70周岁
- 个人征信良好

## 七、禁入/慎入条件

### 7.1 明确禁入
| 情形 | 说明 |
|------|------|
| **企业类型** | 外资企业（部分通道）、港澳台合资、合伙企业（部分产品） |
| **行业禁入** | 房地产、金融投资、高污染、娱乐业（KTV、棋牌室等）、政府机关 |
| **地区禁入** | 部分偏远地区未开放（以系统显示为准） |
| **征信黑户** | 当前逾期、连三累六、冻结/呆账/止付 |
| **司法风险** | 当前涉诉、被执行、失信人名单 |
| **经营异常** | 列入经营异常名录、严重违法失信 |
| **用途违规** | 明确表示用于购房、炒股、还旧贷、固定资产投资等非经营用途 |

### 7.2 慎入/关注
| 情形 | 处理建议 |
|------|----------|
| **征信查询过多** | 近3个月超8次，建议养1-3个月后再申请 |
| **负债率高** | 超70%需补充流水或资产证明 |
| **回款慢/应收账款高** | 需重点核实还款来源 |
| **隐性负债** | 需追问是否有民间借贷、网贷 |
| **频繁变更信息** | 多次修改企业/法人信息需核实原因 |
| **白户** | 可以申请，但额度可能偏低 |

### 7.3 银银合作特殊要求（渣打/星展等）
- 部分银银合作产品要求：中国境内注册满一年的**非外商投资/非外商独资**小微企业
- 每笔总授信额度不超过300万元（银银合作模式）
- 出资方分别为合作银行（70%）和微众银行（30%）

## 八、常见问题应答口径（产品准入类）

### Q1：我能贷多少？
> "微业贷额度由系统根据您企业的纳税记录、开票情况、征信状况、经营流水等多维度数据自动评估，最高可达1000万。具体额度需要您扫码申请后，系统实时测算。我可以帮您看看申请条件是否符合。"

### Q2：利率多少？
> "微业贷年化利率在3.6%-18%之间（单利），按日计息。具体利率由系统根据您的企业信用评级自动定价，审批通过后页面会清楚展示。优质纳税客户有机会享受更低利率。"

### Q3：我能贷多久？
> "借款期限有12期、24期、36期三种，系统根据您的资质自动匹配。优质客户最长可分36期还款。"

### Q4：个体户能申请吗？
> "可以的。微业贷专门为个体工商户开通了申请通道，只要您有正规营业执照、经营满1年以上、有正常纳税记录，法人本人就可以申请。"

### Q5：我刚变更了法人，能申请吗？
> "法人变更后需要等工商信息同步到系统，一般3个工作日左右。如果系统已更新，您可以直接申请。如果提示信息不符，可以联系客服处理。"

### Q6：公司刚成立一年多，能申请吗？
> "企业成立满18个月、且有连续纳税记录，就可以尝试申请。如果未满18个月，建议再经营一段时间，保持良好纳税记录后再申请。"
